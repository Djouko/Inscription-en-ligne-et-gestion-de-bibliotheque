Apres avoir decompresse le dossier de projet contenu dans cet archive,
-ouvrez le dossier dans un terminal
-creez une base de donnees nomme "hms" dans xamp ou wamp ou autre
-tapez la commande pour lancer les migrations et peupler la BD : php artisan migrate --seed
-tapez la commande pour lancer le serveur :   php artisan serve
-Ouvrez un navigateur et saisissez :  http://127.0.0.1:8000

NB: Il s'agit d'une premiere version et donc ce qui a pour l'instant ete pris en compte ce sont juste des livres.
